# Guide Addis Telegram Bot
## Features List:
* Registering Users
* Registering Owners 
* Registering Vehicles 
* Booking
* Search Filters 
     - Vehicle type
* View Car Listings    
* Car Detail  
    - Car Name and Model  
    - Photographs
    - Car Renting Cost per day
* Create Car Listing (For Owners Only)
    - Car owners can list multiple cars
    - Car Basic Details  
    - Photographs  
    - Car Renting Cost per day
* FAQ and Live chat feature
